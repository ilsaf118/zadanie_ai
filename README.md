# Прогнозирование энергопотребления бытовых приборов

## Описание проекта

Проект по прогнозированию суммарного энергопотребления бытовых приборов (стиральная машина, посудомоечная машина, сушилка, холодильник и др.) на основе данных о температуре, влажности внутри помещений и погодных условиях на улице.

Целевая переменная — столбец `Appliances` (энергопотребление в Вт·ч за 10-минутный интервал). Обучение проводилось на классических алгоритмах машинного обучения без использования глубокого обучения.

---

## Данные

**Источник данных:**  
[Appliances energy prediction data](https://github.com/LuisM78/Appliances-energy-prediction-data)  
Оригинальный датасет опубликован в репозитории LuisM78.

**Описание:**  
- Период сбора данных: ~4.5 месяца.  
- Интервал измерений: 10 минут.  
- Данные о температуре и влажности внутри дома собирались через ZigBee-сенсоры (усреднение за 10 минут).  
- Данные об энергопотреблении фиксировались с помощью m-bus счётчиков каждые 10 минут.  
- Погодные данные получены с метеостанции аэропорта Chievres (Бельгия), загружены с rp5.ru и объединены по временной метке.  

**Признаки:**  
- Температура и влажность в 9 комнатах (`T1..T9`, `RH_1..RH_9`).  
- Уличная температура, влажность, давление, скорость ветра, видимость, точка росы.  
- Энергопотребление освещения (`lights`).  

---

## Методы машинного обучения

В проекте реализованы и сравнены следующие алгоритмы:

1. **Linear Regression** — базовый алгоритм для оценки линейной зависимости.  
2. **Random Forest Regressor** — ансамблевый метод на основе решающих деревьев.  
3. **Gradient Boosting Regressor** — градиентный бустинг на деревьях решений.

Выбор оптимальной модели производился на основе метрик:

- **MAE** (средняя абсолютная ошибка)  
- **RMSE** (среднеквадратичная ошибка)  
- **R²** (коэффициент детерминации)

Лучшая модель — Random Forest (настроена с помощью GridSearchCV).  
Итоговые метрики на тестовой выборке:  
- MAE: ~60 Вт·ч  
- RMSE: ~100 Вт·ч  
- R²: ~0.85

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/ваш_аккаунт/appliance-energy-prediction.git
cd appliance-energy-prediction
```

### 2. Создание виртуального окружения

```bash
python3 -m venv venv
source venv/bin/activate   # для Linux/macOS
# venv\Scripts\activate    # для Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск Jupyter Notebook для обучения модели

```bash
jupyter notebook notebooks/energy_prediction.ipynb
```

### 5. Запуск веб-приложения

```bash
streamlit run app/app.py
```

Приложение будет доступно по адресу: `http://localhost:8501`

---

## Использование веб-приложения

В интерфейсе Streamlit необходимо заполнить все поля ввода:

- Освещение (`lights`) — энергопотребление осветительных приборов.
- Температура и влажность в 9 комнатах.
- Уличные погодные условия (температура, влажность, давление, скорость ветра, видимость, точка росы).

После нажатия кнопки «Предсказать» отобразится прогнозируемое значение энергопотребления в Вт·ч.

## Результаты

На тестовой выборке лучшая модель (Random Forest) достигла:

- MAE ≈ 60 Вт·ч  
- RMSE ≈ 100 Вт·ч  
- R² ≈ 0.85

Это означает, что модель объясняет около 85% дисперсии энергопотребления на основе введённых параметров.
